Welcome to SheetHappens

Compare your Excel workbook against a baseline and visualize differences directly in your sheets. All data stays local—no cloud integration required.

Track changes with color-coded highlights and see exactly what's been added, removed, or modified.

Getting Started

Take Snapshot — Save the current workbook as a baseline
Choose File — Upload a .xlsx file to compare against
Baseline dropdowns — Select a saved snapshot or uploaded file
Start Diff — Begin comparing and see changes highlighted
Stop Diff — Remove all highlights and reset colors
Revert Selection — Restore selected cells to baseline values

Understanding Cell Highlights

Green — Added (present now, blank in baseline)
Red — Removed (blank now, present in baseline)
Yellow — Value changed (same formula)
Orange — Formula changed

Select any highlighted cell to see a "New / Old" callout showing the before and after values.
